    新增links页面

    hexo new page links

    打开在/source/links里面刚生成的index.md文件，在头部加上type: "links"，如图：

    代码如下：

    ---
title: 朋友圈
date: 2020-07-19 22:28:52
type: "links"
---

    配置menu

    在主题配置文件menu下添加：

     links: /links/ || fa fa-link

    在\themes\next\languages\zh-CN.yml文件中menu下增加中文描述

    links: 朋友圈

    新增links.swig页面

    在\themes\next\layout新建links.swig，内容如下：

    {% block content %}
  {######################}
  {### LINKS BLOCK ###}
  {######################}

    
    
        
        
    

             
    👨‍🎓 跟着大佬走，成为小大佬
    
            
    
                    {% for link in theme.defaultlinks %}

                    
    
                        
                        
    
                        
     {{ link.nickname }} 
    
                        
    {{ link.info }}
    
                        
    
                    
    

                {% endfor %}

            
    

            
    🍭 五湖四海的朋友们
    

             
    
                    {% for link in theme.friendslinks %}

                    
    
                        
                        
    
                        
     {{ link.nickname }} 
    
                        
    {{ link.info }}
    
                        
    
                    
    

                {% endfor %}

            
    

            {{ page.content }}
            
    
        
    

  {##########################}
  {### END LINKS BLOCK ###}
  {##########################}
{% endblock %}

    修改page.swig

    修改\themes\next\layout\page.swig文件，在开头的block title内部两个elif之间加上：

      {% elif page.type === 'links' and not page.title %}
    {{ __('title.links') + page_title_suffix }}

    如图所示：

    在下面page.content的前面加上：

    {% elif page.type === 'links' %}
           {% include 'links.swig' %}

    为了引入刚才新建的links界面。如图：

    修改links/index.md

    这个是友链页面的申请信息，可以按照自己想法修改：

    ---
title: 朋友圈
date: 2020-07-19 22:28:52
type: "links"
---
### 申请要求：

1、内容持续更新且可以稳定访问
        2、网页整洁无繁杂广告推广
        3、博客主页被百度或谷歌等搜索引擎收录
        4、头像能够快速加载
        5、拥有独立域名

### 友链声明：

1、本站会定期清理无法访问的友链，如果更换了链接信息请至评论区留言，谢谢合作！
        2、本站会定期查看双方是否互为友链，如果取消本站友链，本站也会将您的友链移除

### 申请方式：

先将本站的友链添加到您的友链，相关信息如下
        然后按照以下格式在本站留言区留言，待博主为您添上友链

>名         称：Heson
              头像链接：https://cdn.jsdelivr.net/gh/heson525/pic@master/pic/avatar03.png
              主页链接：https://heson.xyz
              说明信息：人生在勤，不索何获

    修改主题_config配置文件

    在结尾处添加：

    # 友情链接
defaultlinks:
  - nickname: Heson     # 昵称
    avatar: https://cdn.jsdelivr.net/gh/heson525/pic@master/pic/avatar03.png    # 头像地址
    site: https://www.heson10.com #友链地址
    info: 人生在勤，不索何获


friendslinks:
  - nickname: Heson     # 昵称
    avatar: https://cdn.jsdelivr.net/gh/heson525/pic@master/pic/avatar03.png    # 头像地址
    site: https://www.heson10.com #友链地址
    info: 人生在勤，不索何获

    defaultlinks:对应的是links.swig文件中 👨‍🎓 跟着大佬走，成为小大佬段落，此处链接写大佬的博客；
    friendslinks:对应的是links.swig文件中 🍭 五湖四海的朋友们段落，此处链接写朋友的博客。
